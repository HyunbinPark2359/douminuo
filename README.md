# 스마트누오 샘플 포맷터 (Chrome 확장)

## 소개

[스마트누오](https://smartnuo.com) **자체에서 제공하는 샘플 복사** 기능은 기존 커뮤니티에서 주로 사용하는 양식과 달라, 그대로 쓰면 **가독성이 떨어지는 경우**가 많습니다.

이 확장 프로그램은 스마트누오 **공유 URL**만 넣으면, 그 설정을 **짧고 읽기 쉬운 요약 샘플 형식**으로 바꿔 줍니다. 커뮤니티·블로그·메모장 등에 올리기 좋게 **한눈에 들어오는 레이아웃**을 목표로 했습니다.

기본 옵션으로 결과에 **파티·샘플 URL을 함께 붙일 수 있습니다.** 나중에 그 링크를 주소창에 넣거나 스마트누오에서 열면 **다시 불러와서 수정·검토하기**도 수월합니다.

---

## 필요한 것

- **Google Chrome**(또는 Chromium 계열 브라우저)
- 인터넷 연결(확장이 스마트누오 서버에서 공유 데이터를 받아 옵니다)
- **스마트누오에 로그인**한 상태가 필요할 수 있습니다. 비공개·권한이 걸린 공유는 `credentials: 'include'`로 불러오므로, 해당 계정으로 열 수 있는 링크여야 합니다.

---

## 다운로드

1. 공유된 zip 파일을 내려받습니다.
2. zip을 받았다면 **압축을 풉니다.**  
3. 풀린 폴더 안에 `extension` 폴더가 있으면, 설치 시 선택할 폴더는 **`extension`** 입니다.

---

## Chrome에 적용하기 (압축해제된 확장 프로그램)

**방법 1: 주소창에서 이동**

1. Chrome 주소창에 `chrome://extensions` 를 입력해 확장 프로그램 페이지를 엽니다.

**방법 2: 메뉴에서 이동**

1. 크롬 오른쪽 위 점 세 개(더보기) 클릭 → **확장 프로그램** 클릭 → **확장 프로그램 관리**를 클릭합니다.

(방법 1 또는 2로 확장 프로그램 관리 페이지에 들어간 뒤 아래를 따라 합니다.)

2. 오른쪽 위 **개발자 모드**를 켭니다.
3. **압축해제된 확장 프로그램 로드** 를 누릅니다.
4. 압축을 푼 폴더 내의 **`extension`** 폴더를 선택합니다.
5. 툴바에 **샘플 포맷터** 아이콘이 보이면 설치 완료입니다.

업데이트할 때는 소스파일을 새로 받은 뒤, `chrome://extensions`에서 해당 확장의 **새로고침(↻)** 을 누르면 됩니다.

---

## 사용법

1. 툴바에서 아이콘을 눌러 **팝업**을 엽니다.
   - **핀 아이콘**을 누르면 확장 프로그램이 고정되어 더 쉽게 이용할 수 있습니다.
2. **공유 URL** 칸에 스마트누오 샘플 주소를 넣습니다.  
   - 예: `https://smartnuo.com/#ps=xxxxxxxx`
3. 잠시 후 **출력** 칸에 변환 결과가 채워집니다.
4. **복사하기**로 클립보드에 복사합니다.

### URL 입력 규칙

| 입력 | 동작 |
|------|------|
| **한 줄** | 서버 응답으로 **파티**인지 **단일 샘플**인지 자동 판별합니다. 파티면 6칸 전체, 샘플이면 한 마리만 요약합니다. |
| **여러 줄** | 줄마다 **샘플 공유 URL** 하나씩으로 간주합니다. **파티 URL은 여기에 넣으면 안 됩니다.** (파티는 반드시 한 줄만.) |

### 옵션

- **URL**: 결과 맨 위에 파티/샘플 링크를 붙입니다. 끄면 **순수 요약 텍스트만** 나옵니다.
- **실수치**: 스탯 6종 실수치가 모두 있을 때만 `162-143-…` 형태 한 줄을 넣습니다.
- **결정력**: 각 기술 옆에 `기술명 (숫자)` 또는 조건부 특성이 있으면 `기술명 (기본->버프)` 형태로 붙입니다. **재앙의검**은 물리 기술, **재앙의구슬**은 특수 기술 결정력에 ×4/3 근사를 곱해 표시합니다.
- **내구력**: HP·방어·특수방어 **실수치**로 `(HP×방어)/0.411`, `(HP×특방)/0.411` 두 줄을 테라스탈 줄과 기술 줄 사이에 넣고, 도구·특성·아래 확정 규칙만 곱합니다.

### 팁

- 링크가 맞는지 확인하려면 브라우저에서 같은 URL을 한 번 열어 보면 됩니다.
- 팝업에 입력한 내용·체크 상태는 브라우저 **세션** 기준으로 잠깐 기억됩니다(탭을 오래 켜 둔 세션 기준).

---

## 자주 겪는 문제

- **불러오지 못했습니다 / 공유 데이터를 불러오지 못했습니다**  
  → URL에 `#ps=` 가 들어 있는지, 브라우저에서 그 주소가 열리는지, 네트워크·스마트누오 서버 상태를 확인해 보세요.
