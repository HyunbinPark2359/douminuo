# 스마트누오 샘플 변환기 (크롬 확장 프로그램)

## 소개

[스마트누오](https://smartnuo.com/) 팀빌더 자체에서 제공하는 **샘플 복사** 기능은 기존 커뮤니티에서 주로 사용하는 양식과 달라, 그대로 쓰면 **가독성이 떨어지는 경우**가 많습니다.

이 확장 프로그램은 스마트누오 **공유 URL**만 넣으면, 그 설정을 **짧고 읽기 쉬운 요약 샘플 형식**으로 바꿔 줍니다. 커뮤니티·블로그·메모장 등에 올리기 좋게 **한눈에 들어오는 레이아웃**을 목표로 했습니다.

기본 옵션으로 결과에 **파티·샘플 URL을 함께 붙일 수 있습니다.** 나중에 그 링크를 주소창에 넣거나 스마트누오에서 열면 **다시 불러와서 수정·검토하기**도 수월합니다.

추가로 다음을 지원합니다.

- **결정력·내구력** 같은 수치 요약을 출력에 넣기
- **Pokémon Showdown / PokePaste** 스타일 텍스트로보내기(영문 종명·고전 EV 근사 변환)
- 스마트누오 **데미지 계산기**에 공격측·수비측 샘플 URL을 넣어 **폼을 자동으로 채우기**

---

## 필요한 것

- **Google Chrome**(또는 Chromium 계열 브라우저)
- 인터넷 연결(스마트누오 서버에서 공유 데이터를 받습니다)
- **스마트누오에 로그인**한 상태가 필요할 수 있습니다. 비공개·권한이 걸린 공유는 로그인한 계정으로 열 수 있는 링크여야 합니다.
- **결정력** 표시와 **계산기 자동 입력** 일부는 기술 메타 확인을 위해 [PokéAPI](https://pokeapi.co/)에 접속합니다. 해당 기능을 쓸 때는 `pokeapi.co` 접근이 가능해야 합니다.

---

## 다운로드

1. 공유된 zip 파일을 내려받습니다.
2. zip을 받았다면 **압축을 풉니다.**
3. 풀린 폴더 안에 **`extension`** 폴더가 있는지 확인합니다.

---

## Chrome에 적용하기

1. **확장 프로그램** 창을 엽니다.
   1. **방법 1: 주소창에서 이동**
      1. 크롬 주소창에 `chrome://extensions` 를 입력해 확장 프로그램 페이지를 엽니다.
   2. **방법 2: 메뉴에서 이동**
      1. 크롬 오른쪽 위 **점 세 개(더보기)** 클릭
      2. **확장 프로그램** 클릭
      3. **확장 프로그램 관리**를 클릭합니다.
2. 오른쪽 위 **개발자 모드**를 켭니다.
3. 왼쪽 위 **압축해제된 확장 프로그램 로드** 를 누릅니다.
4. 압축을 푼 폴더 내의 **`extension`** 폴더를 선택합니다.
5. **스마트누오 샘플 변환기** 아이콘이 보이면 설치 완료입니다.

업데이트할 때는 새 소스를 덮어쓴 뒤, `chrome://extensions`에서 해당 확장의 **새로고침(↻)** 을 누르면 됩니다.

## 사용법

### 팝업 구성

팝업 상단에는 **샘플 변환**과 **계산기 입력** 두 탭이 있습니다. 기본은 샘플 변환입니다.

### 샘플 변환(요약 텍스트)

1. [스마트누오](https://smartnuo.com/) 웹사이트에 접속합니다.
2. 오른쪽 위 **공유 URL 복사** 버튼을 클릭합니다.
3. 크롬 툴바에서 **샘플 변환기** 아이콘을 눌러 **팝업**을 엽니다.
   - **핀 아이콘**을 누르면 확장 프로그램이 고정되어 더 쉽게 이용할 수 있습니다.
4. **샘플 변환** 탭에서 **공유 URL** 칸에 스마트누오 샘플 주소를 넣습니다.
   - 예) `https://smartnuo.com/#ps=xxxxxxxx`
5. 잠시 후 **출력** 칸에 변환 결과가 채워집니다.
6. **복사하기**로 클립보드에 복사합니다.

#### URL 입력 규칙

- **한 줄:** **파티**인지 **단일 샘플**인지 자동 판별합니다. 파티면 6칸 전체, 샘플이면 한 마리만 요약합니다.
- **여러 줄:** 줄마다 **단일 샘플** 공유 URL 하나씩으로 간주합니다. **파티 URL은 여기에 넣지 마세요.** (파티는 반드시 한 줄만.)

#### 출력 옵션(체크박스)

- **URL:** 결과 맨 위에 **파티/샘플 링크**를 붙입니다. 끄면 순수 요약 텍스트만 나옵니다.
- **실수치:** 스탯 6종 실수치가 모두 있을 때만 `162-143-…` 형태 한 줄을 넣습니다.
- **결정력:** 각 기술 옆에 `기술명 (숫자)` 형태로 붙입니다. 조건부 특성 등으로 위력이 달라지면 `기술명 (기본→버프)`처럼 표시될 수 있습니다. **재앙의검**은 물리 기술, **재앙의구슬**은 특수 기술 결정력에 ×4/3에 가까운 보정을 곱해 표시합니다. (기술·타입 정보는 PokéAPI를 참고합니다.)
- **내구력:** HP·방어·특수방어 **실수치**가 있을 때, 테라스탈 줄과 기술 줄 사이에 `(HP×방어)/0.411`, `(HP×특방)/0.411` 두 줄을 넣습니다. 도구·특성·아래에 정해 둔 규칙만 반영해 곱합니다.

#### Showdown(출력 형식)

**출력** 영역 제목 옆의 **Showdown** 체크를 켜면, 같은 공유 데이터를 **Pokémon Showdown / PokePaste**에 가까운 블록 형식(영문 종명, `Level:`, `EVs:` 등)으로 만듭니다.

- 스마트누오의 노력치(챔피언스형 1:1 노력)를 Showdown의 **고전 EV**(유효 합 508 기준, `floor((EV+4)/8)` 계단에 맞춘 **근사 변환**)로 맞춥니다. 완전히 동일한 배분이 아닐 수 있으나, 같은 “증가 실수치”를 목표로 최소 EV 위주로 맞춥니다.
- 한 줄 입력이 **파티**이면 슬롯마다 블록이 이어집니다.

### 계산기 입력(스마트누오 데미지 계산기)

1. 브라우저에서 스마트누오 **데미지 계산기** 화면을 연 상태로 둡니다.
2. 팝업에서 **계산기 입력** 탭을 엽니다.
3. **공격측**·**수비측** 칸에 각각 **단일 샘플** 공유 URL을 넣습니다. 한쪽만 넣어도 됩니다.
4. **입력** 버튼을 누릅니다.

동작 요약:

- 확장이 URL에서 샘플을 읽고, 열려 있는 스마트누오 탭의 계산기 폼에 종·레벨·노력·기술·도구·특성 등을 채웁니다.
- **파티 공유 URL**은 계산기 입력에 사용할 수 없습니다. 샘플(`#ps=`) URL만 지원합니다.
- 공격 측이 고른 기술이 **특수**로 분류되어 있어도, 피해 계산에 **방어**·**방어 노력치**를 쓰는 기술(예: **사이코쇼크**, **사이코브레이크**, **비밀의칼**)이면 수비측 **방어** 노력치 칸이 채워집니다.
- 기술 분류·이름 확인을 위해 PokéAPI를 사용할 수 있습니다. 응답이 늦거나 없으면 일부가 비어 있거나 경고가 붙을 수 있습니다.

문제가 나면 스마트누오 탭을 **새로고침**한 뒤 다시 시도하거나, 계산기 화면인지 확인해 보세요.

---

## 확장 프로그램 옵션(테마)

`chrome://extensions`에서 이 확장의 **세부정보**를 연 뒤 **확장 프로그램 옵션**을 열면, 팝업의 **밝기(테마)** 를 **시스템 / 라이트 / 다크** 중에서 고를 수 있습니다.

---

## 팁

- 링크가 맞는지 확인하려면 다른 탭에서 같은 URL을 한 번 열어 보면 됩니다.
- 팝업에 입력한 내용·체크 상태·선택한 탭은 브라우저 **세션** 기준으로 잠깐 기억됩니다.

---

## 자주 겪는 문제

- **불러오지 못했습니다 / 공유 데이터를 불러오지 못했습니다**
  - URL에 `#ps=` 가 들어 있는지, 브라우저에서 그 주소가 열리는지, 네트워크·스마트누오 서버 상태를 확인해 보세요.

- **계산기 입력** 관련
  - **파티 공유 URL은 계산기 입력에 사용할 수 없습니다**라고 나오면, 샘플 URL로 바꿔 주세요.
  - **스마트누오 탭을 연 뒤 다시 시도**라고 나오면, `smartnuo.com`이 열린 탭을 둔 상태에서 다시 **입력**을 누릅니다.
  - **데미지 계산기 화면인지 확인**이라고 나오면, 팀빌더가 아니라 계산기 페이지에서 시도해 보세요.
  - **탭과 연결되지 않았습니다**라면 해당 탭을 새로고침한 뒤 재시도합니다.

## 라이선스·면책

스마트누오는 제3자 서비스입니다. 본 확장은 비공식 도구이며, 사이트 정책·API 변경에 따라 동작이 깨질 수 있습니다. Chrome 웹 스토어 미등록 배포 시, 받는 사람에게 위 설치 방법을 함께 안내해 주세요.
