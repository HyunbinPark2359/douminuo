# 스마트누오 샘플 포맷터 (Chrome 확장)

[스마트누오](https://smartnuo.com) **공유 URL**만 넣으면, 짧게 정리된 **요약 샘플 형식** 텍스트로 바꿔 주는 비공식 확장 프로그램입니다. (현재 버전은 베타 느낌의 **0.x** 배포를 가정합니다.)

---

## 필요한 것

- **Google Chrome**(또는 Chromium 계열 브라우저)
- 스마트누오에 **로그인**한 상태(공유 데이터를 불러올 때 쿠키가 필요합니다)
- 불러오려는 공유가 **본인이 접근 가능한** 링크여야 합니다

---

## 다운로드

배포 방식에 따라 아래 중 하나로 받습니다.

- **GitHub**: Releases 등에서 zip을 받거나 저장소를 클론합니다.
- **구글 드라이브 등**: 공유된 zip 파일을 내려받습니다.

zip을 받았다면 **압축을 풉니다.**  
풀린 폴더 안에 `extension` 폴더가 있으면, 설치 시 선택할 폴더는 **`extension`** 입니다. (`manifest.json`이 들어 있는 그 폴더)

---

## Chrome에 적용하기 (압축해제된 확장 프로그램)

1. Chrome 주소창에 `chrome://extensions` 를 입력해 확장 프로그램 페이지를 엽니다.
2. 오른쪽 위 **개발자 모드**를 켭니다.
3. **압축해제된 확장 프로그램을 로드합니다** 를 누릅니다.
4. 폴더 선택 창에서 이 프로젝트의 **`extension`** 폴더를 고릅니다.  
   - `manifest.json`, `background.js`, `popup.html` 등이 한곳에 모여 있어야 합니다.
5. 툴바에 **샘플 포맷터** 아이콘이 보이면 설치 완료입니다.

업데이트할 때는 소스를 새로 받은 뒤, `chrome://extensions`에서 해당 확장의 **새로고침(↻)** 을 누르면 됩니다.

---

## 사용법

1. 툴바에서 확장 아이콘을 눌러 **팝업**을 엽니다.
2. **공유 URL** 칸에 스마트누오 주소를 넣습니다.  
   - 예: `https://smartnuo.com/#ps=xxxxxxxx`  
   - `#ps=` 가 들어 있어야 합니다.
3. 잠시 후 **출력** 칸에 변환 결과가 채워집니다.
4. **복사하기**로 클립보드에 복사합니다.

### URL 입력 규칙

| 입력 | 동작 |
|------|------|
| **한 줄** | 서버 응답으로 **파티**인지 **단일 샘플**인지 자동 판별합니다. 파티면 6칸 전체, 샘플이면 한 마리만 요약합니다. |
| **여러 줄** | 줄마다 **샘플 공유 URL** 하나씩으로 간주합니다. **파티 URL은 여기에 넣으면 안 됩니다.** (파티는 반드시 한 줄만.) |

### 옵션

- **URL 포함** (기본 켜짐): 결과 맨 위에 파티/샘플 링크를 붙입니다. 끄면 **순수 요약 텍스트만** 나옵니다.
- **실능치 포함**: 스탯 6종 실능치가 모두 있을 때만 `162-143-…` 형태 한 줄을 넣습니다.

### 팁

- 같은 브라우저에서 스마트누오에 로그인해 두는 것이 가장 안정적입니다.
- 팝업에 입력한 내용·체크 상태는 브라우저 **세션** 기준으로 잠깐 기억됩니다(탭을 오래 켜 둔 세션 기준).

---

## 자주 겪는 문제

- **불러오지 못했습니다 / 공유 데이터를 불러오지 못했습니다**  
  → 로그인 여부, URL에 `#ps=` 포함 여부, 해당 공유가 열리는지 브라우저에서 먼저 확인해 보세요.

- **지원하지 않는 공유 형식**  
  → 스마트누오 API 응답 형식이 바뀌었을 수 있습니다. 이 경우 `extension/shareToRaw.js` 매핑 수정이 필요할 수 있습니다.

- 코드를 고쳤는데 반영이 안 됨  
  → `chrome://extensions`에서 확장 **새로고침**을 눌렀는지 확인하세요.

## 프로젝트 구조 (개발 참고)

```
nuo-formatter/
├── extension/           ← Chrome에 로드하는 폴더
│   ├── manifest.json
│   ├── background.js
│   ├── shareToRaw.js
│   ├── formatter.js
│   ├── popup.html / popup.js / popup.css
│   └── …
├── formatter.js         ← 포맷 로직 원본 (extension/formatter.js 와 동기화)
└── BACKLOG.md
```

포맷 규칙을 바꿀 때는 보통 루트 `formatter.js`를 수정한 뒤 `extension/formatter.js`에 같은 내용을 맞춥니다.
