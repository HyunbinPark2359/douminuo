## Showdown 붙여넣기 — PokeAPI 한글 없음 폴백

**원인:** PokeAPI `names`에 `language: ko`가 없으면 `moveKoMap.json` / `itemKoMap.json` 생성 시 해당 항목이 빠짐 → 한글 그대로 출력.

**처리:** `extension/moveKoFallback.json`, `itemKoFallback.json`에 **누오에 보이는 한글 키 → Showdown id / item slug** 를 넣고, 팝업 로드·`generate-move-ko-map.js`에서 **기존 맵에 없을 때만** 덧씀.

**반영됨 (유지·검증만):** `찬물끼얹기`→`chillingwater`, `라이트닝드라이브`→`electrodrift`, `펀치글러브`→`punching-glove` 는 폴백 JSON에 들어 있음. 공식 한글 명칭과 다르면 키를 고치고, PokeAPI에 `ko`가 생기면 해당 폴백 줄은 **삭제**해도 됨.

**전수 갱신:** `node scripts/audit-pokeapi-ko-gaps.js` → `scripts/pokeapi-ko-gaps-report.json` (갭 요약).

**PokeAPI 본맵 일괄 재생성:** `node scripts/refresh-ko-maps-from-pokeapi.js`  
→ `moveTags.json` → `moveKoMap.json` → `itemKoMap.json` → `abilityKoMap.json` → `moveSlugToEn.json` (순서 고정, 네트워크·수분 소요).

개별만: `generate-move-tags.js` / `generate-move-ko-map.js` / `generate-item-ko-map.js` / `generate-ability-ko-map.js` / `generate-move-slug-to-en.js`.

누오에서 한글로 튀면 폴백에 **누오 표기 → id/slug** 추가: `moveKoFallback.json`, `itemKoFallback.json`, `abilityKoFallback.json`.

- [ ] **Showdown 샘플 → 스마트누오 공유 URL** — Showdown 붙여넣기 텍스트를 입력하면 `#ps=` 등 **스마트누오 샘플 URL**로 바꿔 주는 기능(현재 흐름의 역방향). 아래 **공유 API 메모**로 페이로드 생성 가능 여부를 판단.

**메모 — 샘플 공유 URL 버튼 시 Network (역방향·URL 조립용):**

- **Request:** `POST https://smartnuo.com/api/party/share`
- **Response:** `{"id":"FIuWr95U"}` — 짧은 id 한 개. 실제 브라우저 주소줄의 `#ps=`(또는 동일 역할 파라미터)에 이 값이 들어가는지는 누오 UI에서 한 번 더 확인하면 됨. -> 들어가는것 확인.
- **Payload 골격:** 최상위 `params` → `data` → `{ "slot": <0–5>, "data": { … } }`.
  - `data`(슬롯 페이로드): `sampleName`, `nameKr`, `movesKr`(한글 기술명 배열), `pokemon` 객체.
  - `pokemon`: `name`(영문 종 slug, 예: `corviknight`), `sprite`(PokeAPI URL), `level`, `personality`, `terastal`, `ability`, `equipment`, `moves`(각 `{ name, type, power, damage_class }` — `name`은 `body-press` 등 slug), `stats`(종별 `hp`/`attack`/… 각 `value`·`real`), `firstType`, `baseStats`(종족값 정수).
- **예시(캡처 원문):** 슬롯 `4`, 샘플명 `물리접수 아머까오`, 위 구조로 아머까오 한 마리 전체가 실려 있음. 전체 JSON은 네트워크 탭 복사본을 보관할 것(할일.md에는 구조만 적음).
- **구현 시 확인:** 동일 POST를 확장/스크립트에서 보낼 때 **쿠키·세션·CORS** 필요 여부, `manifest.json`의 `host_permissions`만으로 충분한지.
