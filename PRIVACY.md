# 도우미누오 개인정보처리방침

**최종 수정일: 2026-04-20**
**대상 제품: 도우미누오 (Chrome 확장 프로그램)**

도우미누오(이하 "본 확장")는 사용자의 개인정보를 존중하며, 꼭 필요한
경우에만 최소한의 데이터를 사용합니다. 본 개인정보처리방침은 본 확장이
어떤 데이터를 어떻게 다루는지 투명하게 안내하기 위해 작성되었습니다.

## 1. 수집·이용하는 정보

본 확장은 사용자의 이름, 이메일, 연락처, 위치, 결제 정보, 건강 정보,
인증 정보, 웹 기록, 마우스·키 입력 등 **개인 식별에 사용될 수 있는
어떠한 정보도 수집하지 않습니다.**

다만 본 확장의 핵심 기능을 수행하기 위해 다음과 같이 제한된 범위의
데이터를 로컬에서 **사용(read)** 합니다.

### 1-1. 스마트누오 웹사이트 콘텐츠

- **대상**: 사용자가 스마트누오(https://smartnuo.com) 팀빌더 또는 데미지
  계산기 페이지에 직접 입력·구성한 파티·샘플 정보(포켓몬 종족, 기술,
  특성, 지닌물건, 노력치, 개체값, 레벨, 성격 등) 및 사용자가 직접
  붙여 넣은 공유 URL(`https://smartnuo.com/#ps=...`).
- **이용 목적**:
  - 해당 파티·샘플을 읽기 쉬운 텍스트 형식으로 변환하여 사용자의
    클립보드에 복사해 주는 기능
  - 데미지 계산기 페이지의 입력 폼(종족, 노력치, 기술 등)을 자동으로
    채워 주는 기능
  - 팀빌더 화면에 각 포켓몬의 결정력·내구력을 실시간으로 계산하여
    표시해 주는 기능
- **처리 위치**: 모든 처리는 사용자의 브라우저 내부에서만 이루어집니다.
- **외부 전송 여부**: 없음. 개발자를 포함한 어떠한 제3자에게도 전송되지
  않습니다.
- **보관 기간**: 본 확장은 해당 데이터를 디스크나 영구 저장소에 기록하지
  않으며, 오직 브라우저의 **임시 메모리(RAM) 상에서만** 처리된 후 즉시
  휘발(폐기)됩니다.

### 1-2. 사용자 환경설정

- **대상**: 팝업 테마(라이트/다크/시스템), 플로팅 버튼 표시 여부(계산기·
  팀빌더 각각), 팀빌더 인라인 결정력·내구력 표시 여부.
- **저장 위치**: `chrome.storage.local`(사용자의 브라우저 내부 저장소).
- **외부 전송 여부**: 없음.
- **보관 기간**: 사용자가 확장을 제거하거나 브라우저 데이터를 삭제하기
  전까지 유지됩니다.

### 1-3. 포켓몬 공개 메타데이터 캐시

- **대상**: 포켓몬 기술·아이템·특성의 한국어 표기 등, PokéAPI
  (https://pokeapi.co)에서 조회한 공개 도감 정보.
- **이용 목적**: 매번 동일한 외부 요청을 반복하지 않기 위한 로컬 캐시.
- **저장 위치**: `chrome.storage.local`.
- **개인정보 포함 여부**: 없음. 사용자와 무관한 공개 정보만 저장합니다.

## 2. 외부 통신

본 확장은 기능 수행을 위해 다음 두 개의 도메인에만 네트워크 요청을
보냅니다. 두 요청 모두 **사용자 식별 정보(계정, 토큰, 쿠키 외 추가
헤더 등)를 일절 포함하지 않습니다.**

- **https://smartnuo.com / https://www.smartnuo.com**
  사용자가 직접 제공한 공유 URL로부터 파티·샘플 데이터(JSON)를
  조회하기 위해 사용합니다.

- **https://pokeapi.co**
  포켓몬 타입·기술·아이템의 한국어 표기 등 공개 도감 정보를
  조회하기 위해 사용합니다. PokéAPI는 인증이 필요 없는 공개 API입니다.

## 3. 쿠키 및 추적 기술

본 확장은 자체적으로 쿠키를 생성하거나, 사용자의 활동을 추적하기
위한 어떠한 분석 도구(Google Analytics, Firebase 등)도 포함하고
있지 않습니다.

## 4. 제3자 제공 및 판매

본 확장은 위 1항에 명시된 어떠한 데이터도 **제3자에게 판매·양도·전송
하지 않습니다.** 또한 신용도 판단, 대출 심사, 광고 타기팅 등 본 확장의
전용 목적과 무관한 어떠한 용도로도 사용하지 않습니다.

## 5. 어린이 개인정보 보호

본 확장은 특정 연령대를 대상으로 하지 않으며, 만 13세 미만 아동으로
부터 개인정보를 의도적으로 수집하지 않습니다. 앞서 기술한 바와 같이
본 확장은 연령을 포함한 어떠한 개인 식별 정보도 수집하지 않습니다.

## 6. 사용자 권리 및 데이터 삭제

본 확장이 로컬에 저장하는 데이터는 환경설정 값과 PokéAPI 캐시뿐
이며, 다음 방법으로 완전히 삭제할 수 있습니다.

1. `chrome://extensions` 페이지에서 도우미누오를 제거합니다.
2. 또는 크롬 설정 → "개인정보 및 보안" → "인터넷 사용 기록 삭제"
   에서 "쿠키 및 기타 사이트 데이터" 항목을 포함해 데이터를 지웁니다.

어느 쪽이든 본 확장이 저장한 모든 로컬 데이터가 제거됩니다.

## 7. 면책

본 확장은 스마트누오(https://smartnuo.com) 운영진과 무관한 비공식
(3rd-party) 도구입니다. 스마트누오 측과 어떠한 제휴·협력·후원
관계도 없습니다.

스마트누오 웹사이트 자체의 개인정보 처리에 대해서는 스마트누오의
공식 정책을 별도로 확인해 주세요. 본 방침은 오직 "도우미누오" 확장
프로그램의 동작에 한해 적용됩니다.
