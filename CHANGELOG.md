# 패치 노트

## [1.3.0] - 2026-05-11

### 변경

- **스마트누오 사이트 리뉴얼 (2026-05-04 패치)** 에 맞춰 도우미누오 전체 기능을 복구했습니다.

## [1.2.0] - 2026-04-27

### 새 기능

- **계산기 페이지의 플로팅 메뉴를 리뉴얼 했습니다.**
  - **파티의 포켓몬**을 **클릭 한 번**으로 자동으로 계산기에 입력할 수 있습니다. 공유 URL 직접 입력 역시 지원합니다.
  - 플로팅 메뉴를 펼치고 있는 동안 공격측 또는 수비측 계산기에 **하이라이트**가 켜집니다. 지금 둘 중 어느 패널에 적용될지 쉽게 확인할 수 있습니다. 설정에서 끌 수 있습니다.

### 그 외

- 트래픽 및 성능을 최적화하였습니다.

## [1.1.0] - 2026-04-22

### 새 기능

- **간단 스피드 계산기**를 추가했습니다.
  - 포켓몬의 **추월 범위**를 보여 주는 패널을 추가했습니다.
  - **팀빌더 페이지 우측 패널**에서 확인할 수 있습니다.
  - **스피드 표**를 통해 어느 포켓몬까지 추월하는지 쉽게 확인할 수 있습니다.

### 그 외

- 환경설정 창에 **간단 스피드 계산기** 관련 설정을 추가했습니다.
- Chrome 웹 스토어 **상점 페이지**에 간단 스피드 계산기 이미지를 추가했습니다.
- Chrome 웹 스토어 **상점 페이지 설명란**에 간단 스피드 계산기 기능 안내를 추가했습니다.
